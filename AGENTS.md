## Key Conventions

- **ES Modules**: imports use `.js` extension
- **UI**: Use `src/presentation/ui/` helpers, never raw `console.log`
- **Errors**: Tools return error objects (don't throw)
- **Security**: Content sanitized via `utils/sanitizer.ts`
