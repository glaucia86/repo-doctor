Get running in under 5 minutes, then scale to deep analysis and issue publishing.

Node >= 24Copilot SDK poweredIssue publishing supported

Global Install

repo-doctor analyze owner/repo
View docs

From Source

git clone https://github.com/glaucia86/repo-doctor.git
cd repo-doctor
npm install
npm run build
npm run dev:cli
